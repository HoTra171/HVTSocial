# HVTSocial

Dự án mạng xã hội HVTSocial với Backend (Node.js) và Frontend (React).

## Cấu trúc dự án

- **Backend**: Node.js/Express server
- **Frontend**: React + Vite

## Cài đặt

### Backend
```bash
cd Backend
npm install
```

### Frontend
```bash
cd Frontend
npm install
```

## Chạy dự án

### Backend
```bash
cd Backend
npm start
```

### Frontend
```bash
cd Frontend
npm run dev
```

## Yêu cầu

- Node.js
- npm hoặc yarn

## Môi trường

Tạo file `.env` trong thư mục Backend và Frontend theo mẫu:
- Backend/.env
- Frontend/.env
